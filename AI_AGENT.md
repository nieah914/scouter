### 코드작성 요청시

1. 요청사항에 대해서 plan.md에 Todo List로 작성해서 진행할 계획서를 작성해
2. 작성한 계획서를 기반으로 코드를 작성해
3. 작성한 코드를 테스트하고 리팩토링해
4. 작성한 코드를 문서화하고 문서를 테스트해
5. 작성한 문서를 리뷰하고 수정해
6. 작업이 완료가 되면 history.md 에 진행한 내역에 대한 부분을 기록 해줘
7. 작업 완료 후 todolist.md 에서 해당 항목을 ✅ 완료 처리해줘
